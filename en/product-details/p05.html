MTT-CP.TR: Remote Monitoring
and Control System
for CP Rectifiers

The CP Rectifier Telemetry System for automated data acquisition, storage, and transmission of CP parameters to a central dispatching system

A single 1K-T controller supports up to five CP Rectifiers.

System Composition

Hardware: Software:
Power supply / solar panel (optional) Database
GSM modem for data transmission Monitoring software
1K-T telemetry controller (connection of up to 5 CP Rectifiers) User web interface
Data collection and processing server

Functions
  • Acquisition of pipe-to-soil potential, current,
    and voltage measurements from CP Rectifiers
  • Automatic data transmission via GSM
    communication to the central server
  • Collection, processing, archiving, and
    transmission of data for further analysis
  • Generation of data archives for monitoring
    the condition of ECP systems
